# SoloOS v0.2 产品需求文档

## 定位

SoloOS v0.2 从简单项目表单升级为个人创业者的统一项目管理中心，集中管理 AI 主题商店、百家号自动化、AI 动画 IP、AI 赚钱机会发现、自媒体项目、Agent 软件和未来新项目。

## 参考产品

- Notion：结构化信息管理。
- Trello：项目看板和状态流转。
- Linear：项目列表、优先级和执行节奏。

SoloOS 不复制团队协作软件，而是优先服务个人创业者的一人公司经营。

## v0.2 优先交付

1. 项目总览：首页展示项目统计、今日重点和最近活动。
2. 项目列表：支持搜索、排序、筛选，左侧显示真正项目列表。
3. 项目看板：按状态列展示项目，支持拖拽修改状态。
4. 项目详情：展示基础信息、当前目标、下一步行动、任务、决策、聊天记录、AI 建议、评分和时间轴。
5. 任务系统：任务支持待办、进行中、已完成。
6. 决策记录：记录为什么开始、暂停、放弃或继续。
7. 时间轴：自动记录创建项目、修改状态、新增任务、完成任务等关键事件。

## 数据原则

- 所有数据本地保存。
- v0.2 继续使用 `data/projects.json`，保持可读、可 Git 追踪、可迁移。
- 后续数据规模变大后迁移到 SQLite。
- 删除项目前必须二次确认。

## 目标体验

打开 SoloOS 后，Owner 能一眼看到：

1. 有哪些项目。
2. 哪些项目正在推进。
3. 下一步该做什么。
4. 哪个项目最值得继续投入。
